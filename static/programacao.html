/* ======================================== CONTROLES DE TAMANHO DA SEÇÃO PRIORITY-LINES ======================================== Para ajustar o tamanho da seção "Chamados em Aberto": 1. ALTURA TOTAL DA SEÇÃO: .priority-lines { max-height: 300px; } - Valores menores = seção mais compacta - Valores maiores = seção mais espaçosa - Exemplos: 200px (muito compacto), 400px (espaçoso) 2. ALTURA DE CADA LINHA: .priority-line { min-height: 50px; } - Valores menores = linhas mais baixas - Valores maiores = linhas mais altas - Exemplos: 40px (compacto), 70px (espaçoso) 3. ESPAÇAMENTO INTERNO: .priority-line { padding: 10px 15px; } - Primeiro valor = espaçamento vertical - Segundo valor = espaçamento horizontal - Exemplos: 5px 15px (compacto), 15px 20px (espaçoso) 4. ESPAÇAMENTO ENTRE LINHAS: .priority-lines { gap: 8px; } - Valores menores = linhas mais próximas - Valores maiores = linhas mais separadas - Exemplos: 5px (próximas), 15px (separadas) 5. LARGURA DOS LABELS: .priority-label { min-width: 100px; } - Controla a largura dos textos "PRIORIDADE ALTA", etc. - Exemplos: 80px (compacto), 140px (espaçoso) ======================================== CONTROLES DE ALINHAMENTO DOS CARDS ======================================== Para manter os cards alinhados com as linhas de prioridade: 1. ALTURA DOS CARDS: .chamado-card { height: 50px; } - DEVE ser igual ao min-height das priority-lines - Se mudar priority-line para 60px, mude card para 60px também 2. PADDING DOS CARDS: .chamado-card { padding: 8px 12px; } - Ajustar conforme a altura disponível - Altura 40px = padding 6px 12px - Altura 60px = padding 12px 15px 3. TAMANHO DA FONTE NOS CARDS: .chamado-id { font-size: 10px; } .chamado-descricao { font-size: 11px; } .chamado-info { font-size: 9px; } - Reduzir se cards ficarem muito apertados - Aumentar se houver espaço sobrando */
